GATv2 regressor model for node-level flow-loss prediction.

Two-layer GATv2 network that predicts a scalar value per node.

Architecture¶

Layer 1 : GATv2Conv(in → hidden, heads=H, concat=True) → LayerNorm → ELU → Dropout Layer 2 : GATv2Conv(hidden*H → hidden, heads=H, concat=False) → LayerNorm → ELU → Dropout Layer 3 : GATv2Conv(hidden*H → hidden, heads=H, concat=False) → LayerNorm → ELU Head : Linear(hidden → 1) → squeeze

Edge features are forwarded through both conv layers via edge_dim.

Run the forward pass and return per-node scalar predictions.

During training, applies DropEdge to edge_index (and slices edge_attr accordingly). The output is the squeezed scalar head (no activation), suitable for regression.
